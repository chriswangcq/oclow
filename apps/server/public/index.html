MCP

Connect Agent

Connect AI Meditations to an MCP-capable client so agents can read and update your workspace through the file sandbox rules.

Start with what your client supports Use automatic OAuth when the client has an OAuth, authorization, or connect flow. Use manual JSON only when that is all the client supports.

Recommended

Automatic OAuth

Copy this address into your client's MCP Server URL field. The client will open a browser; sign in and approve access.

-
  1. Choose OAuth or Remote MCP in your client.
  2. Paste the MCP Server URL above.
  3. Follow the prompt to sign in and approve access.

Fallback

Manual configuration

If your client does not support OAuth, copy the full JSON config. It contains a Bearer key for this account, so only use it with clients you trust.
